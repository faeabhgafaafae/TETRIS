# Modern Tetris Web Game

A modern implementation of Tetris built with HTML5 Canvas and JavaScript (ES6+), following best practices like the 7-bag randomizer and `requestAnimationFrame` for smooth gameplay.

## How to Play
- **Arrows Left/Right**: Move piece
- **Arrow Down**: Soft drop
- **Arrow Up**: Rotate piece
- **Space**: Hard drop
- **Play Button**: Start game

## Deployment to GitHub Pages
To deploy this game to GitHub Pages:
1. Push this code to your GitHub repository.
2. Go to your repository settings on GitHub.
3. Select "Pages" on the left menu.
4. Under "Build and deployment", set "Source" to "Deploy from a branch".
5. Choose the branch (e.g., `main`) and the folder (e.g., `/root`).
6. Click "Save".

The game will be available at: `https://<your-username>.github.io/<repository-name>/`
For this repository: `https://faeabhgafaafae.github.io/TETRIS/`
